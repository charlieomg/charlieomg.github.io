使用 Firefox，掌控你的上網
無論你想用偏好的搜尋引擎、瀏覽更乾淨的網頁，或需要一款穩定又快速的瀏覽器，Firefox 每次開啟都帶來速度、安全與簡潔。登入 Mozilla 帳號後，你的密碼、瀏覽紀錄與隱私設定可在各裝置間安全同步，走到哪裡用到哪裡。
主要功能
隱私與安全（預設即保護）
• 自動封鎖追蹤：預設封鎖社交媒體追蹤器、跨網站 Cookie 追蹤、加密貨幣挖礦指令碼，以及指紋辨識（fingerprinting）等。
• 強化追蹤保護：選擇「嚴格」模式，可進一步限制追蹤。
• 私密瀏覽模式：在私密分頁中搜尋與瀏覽，關閉後會自動從本機刪除瀏覽紀錄。
• 鎖定私密分頁：離開時自動上鎖，須以臉部辨識、指紋或 PIN 碼解鎖方可查看。
更快的瀏覽體驗
• 強化追蹤保護會阻止廣告追蹤器在網路上跟著你、拖慢頁面載入，讓上網更俐落。
高效分頁與多裝置協作
• 隨開隨用：想開多少分頁都可以，也不凌亂。
• 快速定位頁面：用縮圖或清單兩種視圖。
• 多端無縫銜接：登入 Mozilla 帳號後，手機與電腦可互看彼此的分頁。
密碼與登入更省心
• 自動記錄與填寫：登入 Mozilla 帳號後，可在多裝置安全同步常用網站的密碼，一鍵登入。
• 強密碼建議：新註冊時可自動產生並安全儲存強密碼。
更懂你的搜尋
• 智能聯想：搜尋列提供建議與歷史結果，直達常用網站。
• 單手更方便：可將搜尋列由頂端移至底端，大拇指更好操作。
• 主畫面小工具：在手機主畫面直接用 Firefox 搜尋。
• 跨裝置最近搜尋：手機與電腦之間的近期搜尋無縫銜接。
• 搜尋引擎可選：可將你偏好的搜尋引擎設為預設；在私密模式中同樣有效。
個人化與擴充套件
• 支援實用的擴充套件（如廣告攔截、封鎖特定網站、加強隱私設定等），打造你的專屬瀏覽體驗。
深色模式 省電護眼
• 隨時切換深色模式，減輕視覺疲勞，延長電池續航力。
多工看影片
• 支援畫中畫（小窗置頂）或背景播放，邊看／聽邊處理其他工作，不中斷。
便捷分享
• 更少步驟，將網頁或頁面重點內容快速分享至常用應用程式。
• 不論是否在私密／無痕模式下，分享同樣安全。

Firefox 由 Mozilla 發起，並由全球開源社群共同打造與維護。我們堅持獨立與非營利，以用戶為先，持續讓網際網路——以及你在網路上度過的每一分鐘——更開放、更安全、更值得信任。

FAQ
  • 什麼是 CloudMoon？

    CloudMoon 是一款雲端遊戲 / 遊戲串流服務，讓你無需下載就能在線上暢玩遊戲。只需一個 15 MB 的 App，或透過瀏覽器，即可在 Android、iOS、平板、Chromebook、低階筆電甚至舊款手機上啟動超過 200 款手機與電腦 / AAA 級遊戲（如《原神》、《Fortnite》、《Roblox》等）。遊戲畫面由我們全球的 GPU 雲端伺服器即時運算（「行動雲遊戲 / 雲手機遊戲」）並串流到你的裝置上。

  • 為什麼選擇 CloudMoon，而不是本地安裝遊戲？

    ・釋放儲存空間 / 不佔用存儲：即使是 32 GB 的手機也能暢玩 200+ 款遊戲，所有遊戲都在雲端運行。
    ・低階手機也能跑高畫質遊戲：3–4 GB RAM 裝置可串流至 60 fps，不卡頓、不過熱。
    ・即點即玩：免下載 10 GB+ 遊戲內容與首日補丁包。
    ・平價雲端遊戲：月費只要150台币，比購買新硬體或使用 Xbox Cloud Gaming、Geforce Now、PlayStation Plus、Now.gg 更划算。
    ・零延遲、不當機：22 個地區邊緣伺服器將延遲降至 55ms 以下。

  • CloudMoon 安全嗎？

    是的。我們將 CloudMoon 設計為「全時啟用、零信任架構」的平台。所有資料傳輸均採用 TLS 1.3 加密；每位用戶的遊戲會話都在獨立的沙盒虛擬機中執行以確保隔離。目前我們已在超過 150 個國家為付費玩家提供服務，未曾發生任何資料外洩事件。

  • 為什麼有時候 HD 看起來像 SD？

    HD 提高了串流的位元率，所以在畫面移動時會更清晰。但在靜止畫面下，HD 和 SD 看起來可能差不多。如果整體畫面都模糊，請檢查你的網路穩定性：雲端遊戲比 YouTube 更需要穩定的連線品質。

  • 我網路很快，為什麼還是會延遲或卡頓？

    速度快不代表穩定 — 低延遲和穩定的 ping 更重要。使用 BT、下載影片，或 Wi-Fi 擁擠都可能造成抖動。建議切換到 5G 網路或使用有線連接，並暫停大型下載。

  • 我可以推薦想玩的遊戲嗎？
