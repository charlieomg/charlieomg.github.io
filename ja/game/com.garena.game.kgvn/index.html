特徴
1. モバイルで多彩なゲームモードをプレイ。自由に対戦してスキルを披露しよう！
5v5マップに加え、ランダム5v5、3v3、1v1など、友達とカスタマイズ可能なマップを自由にプレイできます。他にも、Garena Co Lien Quan/Lien Quanチェスモバイルのように、超高速でアップデートされる、非常に面白くて無料のゲームモードも多数用意されています。
2. 絶え間ない戦い、最高レベルのスキルで勝負
真のMOBA eスポーツスタイルで絶対的な公平性を保証するために、勝利はスキル、チームメイト、そして戦術という3つの重要な要素に基づいています。ゲーム内でGarenaパブリッシャーのアライアンスを選択し、ジャングルに潜んで青と赤のチャームを奪い、相手の動きをブロックして素早く戦いましょう。あなたのスキルで伝説を築こう！
3. いつでもどこでもモバイルゲームをプレイ。完全無料で対戦可能
超高速で最大pingのモバイルゲームをダウンロード。瞬く間に対戦相手を見つけて対戦を開始できます。 5対5のバトルフィールドまたはオートチェスモード（連闊チェス）で、腕前を披露し対戦相手をブロックスできます。1試合わずか10分。いつでもどこでも、Garena Lien Quan Mobileを自由にプレイできます。超高速の連闊バトルは完全無料！
4. モバイルですぐに使える簡単操作
ゲーム操作システムはミニマルで洗練されており、モバイルでは2つの仮想キーボードのみを使用します。プレイヤーは2本の指だけで操作でき、わずか5秒でGarena Lien Quan Mobileを体験できます。あらゆるバトルに挑み、対戦相手をブロックスできます！超高速で操作も簡単です。
5. 便利なボイスチャット、モバイルでの簡単コミュニケーション
リアルタイムボイスチャットとモバイルでのクイックチャットを統合したシステムで、柔軟かつ自由にカスタマイズできます。試合の進行が速いほど、通信速度も速く便利になります。クラン同盟とのコミュニケーションにはボイスチャットを必ず使用してください。5v5 Garena Lien Quan Mobile の戦場に参加する際は、ブロックされることを恐れないでください。
6. ランク戦に参戦し、最高峰に挑め
バンバン… ラウンド終了時に数発の射撃で敵チームを完全にブロックし、バトルの勝敗を決定づけることができます。MOBA eスポーツ満載の熾烈なランク戦に参戦し、Garena Lien Quan Mobile の伝説的なランキングであなたの勇気を示しましょう。超高速ランク戦を見つけ、超高速でLien Quanと戦いましょう。

FAQ
  • CloudMoon とは何ですか？

    CloudMoon は、ダウンロード不要でオンラインゲームをプレイできるクラウドゲーミング / ゲームストリーミングサービスです。15 MB のアプリ、またはブラウザだけで、Android、iOS、タブレット、Chromebook、低スペックのノートPCや旧型スマートフォンでも『原神』、『フォートナイト』、『ロブロックス』など、200 本以上のモバイル & PC / AAA ゲームをすぐにプレイできます。グラフィックスは世界中の GPU サーバーで処理され（「モバイルクラウドゲーミング / クラウドフォンゲーミング」）、リアルタイムであなたのデバイスにストリーミングされます。

  • なぜローカルインストールではなく CloudMoon を選ぶのか？

    ・ストレージ節約 / メモリ不要：32GBのスマホでも200本以上のゲームがプレイ可能—すべてクラウド上にあります。
    ・低スペック端末でも高グラフィックゲームが可能：3〜4GBのRAMで最大60fpsまでストリーミング—発熱なし。
    ・即プレイ：10GB以上のダウンロードや初日パッチは不要。
    ・お手頃なクラウドゲーミング：月額800円から—新しいハードやXbox Cloud Gaming、Geforce Now、PS Plus、Now.ggよりずっと安い。
    ・ラグなし・クラッシュなし：22拠点のエッジサーバーで遅延は55ms以下に。

  • CloudMoon は安全ですか？

    はい。CloudMoon は「常時稼働・ゼロトラスト」設計のプラットフォームです。通信は TLS 1.3 により暗号化されており、各ユーザーセッションは独立したサンドボックス仮想マシンで実行され、完全に分離されています。現在、150 以上の国で有料ユーザーにサービスを提供しており、これまでに一度もデータ漏洩は発生していません。

  • なぜHDがSDのように見えることがあるの？

    HDはストリーミングのビットレートを上げ、動きのある映像をより鮮明にします。静止画面ではHDとSDの違いが目立ちにくくなります。全体的にぼやけて見える場合は、回線の安定性を確認してください。クラウドゲーミングはYouTubeよりも安定した接続を必要とします。

  • 高速なインターネットなのに、なぜラグやカクつきが起きるの？

    速度だけでは不十分です — 低レイテンシーと安定した ping のほうが重要です。トレントや動画のダウンロード、混雑したWi-Fiはジッターの原因になります。5GHz 帯への切り替えか有線接続にして、大容量のダウンロードは一時停止しましょう。
